# LetsDefend WriteUp

Recueil de write-up des alertes SOC traitées sur la plateforme [LetsDefend](https://letsdefend.io), dans le cadre d'un entraînement pratique en analyse SOC (Blue Team / DFIR).

Chaque write-up suit une structure commune : contexte, timeline, analyse détaillée, IOC, verdict, actions et recommandations. Voir [template.md](template.md).
